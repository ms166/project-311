# Installation
* Install python 3 on your system
* Create a virtual environment in any directory:
  * Unix: `python3 -m /path/to/new/virtual/environment` 
  * Windows: `c:\>c:\Python35\python -m venv c:\path\to\myenv`
  * For more info: https://docs.python.org/3/library/venv.html
